# React Examples With Webpack

* Run Prettier Command

`npm run format`

* Build the application
`npx webpack`

(or)

`npx webpack --config webpack.config.js`

`npx build`

* Watch the application

`npm watch`

* Run the application

`npm run web_start`

* Run server

`npm run w_server`
